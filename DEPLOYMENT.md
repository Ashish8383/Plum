# Deployment Guide - Shiv Shankar Plumber Website

## Quick Start

Your premium plumbing website is **ready for production**! Follow these steps to deploy.

---

## Option 1: Deploy to Vercel (Recommended)

### Step 1: Create GitHub Repository
1. Go to [GitHub.com](https://github.com)
2. Create a new repository named `shiv-shankar-plumber`
3. Clone locally and push your code:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Shiv Shankar Plumber website"
   git remote add origin https://github.com/YOUR_USERNAME/shiv-shankar-plumber.git
   git push -u origin main
   ```

### Step 2: Deploy on Vercel
1. Go to [Vercel.com](https://vercel.com)
2. Sign in with GitHub
3. Click "New Project"
4. Select your `shiv-shankar-plumber` repository
5. Click "Deploy"
6. Wait for deployment (takes ~2-3 minutes)

### Step 3: Get Your Live URL
- Your site will be available at: `https://shiv-shankar-plumber.vercel.app`
- You can set a custom domain in Vercel settings

---

## Option 2: Deploy to Other Platforms

### Netlify
1. Connect GitHub repository to Netlify
2. Build command: `pnpm build`
3. Publish directory: `.next`
4. Deploy automatically

### Heroku
1. Create `Procfile` with: `web: pnpm start`
2. Add `buildpacks` for Node.js
3. Deploy via git push

### Self-Hosted (AWS, DigitalOcean, etc.)
1. Build locally: `pnpm build`
2. Use Node.js production server
3. Ensure 3000 port is open

## Domain Setup

### Option A: Use Vercel Domain (Free)
- Vercel provides free `*.vercel.app` subdomain
- Available after deployment

### Option B: Custom Domain (Recommended)
1. Buy domain from GoDaddy, Namecheap, or Google Domains
2. In Vercel Settings → Domains
3. Add your domain
4. Follow DNS setup instructions (usually 5-10 minutes to propagate)
5. Add SSL certificate (automatic with Vercel)

## Troubleshooting

### Site Not Loading After Deploy
1. Check Vercel deployment logs
2. Clear browser cache (Ctrl+Shift+Del)
3. Try different browser or incognito mode
4. Wait 5 minutes for DNS propagation

### WhatsApp Links Not Working
1. Verify phone numbers have country code (+91)
2. Test on mobile device (works best on mobile)
3. Ensure WhatsApp is installed on test device
4. Try on desktop with WhatsApp Web

### Slow Loading
1. Check image URLs (Unsplash should be fast)
2. Monitor Vercel Analytics
3. Run Lighthouse audit
4. Check Core Web Vitals

### Styling Issues
1. Clear browser cache
2. Hard refresh page (Ctrl+Shift+R)
3. Check Tailwind CSS is loaded
4. Verify browser supports CSS Grid/Flexbox
